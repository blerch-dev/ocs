build docker images, load them into minikube, specify 'imagePullPolicy: Never' in deployment.template.spec
need load balancer like

apiVersion: v1
kind: Service
metadata:
  name: kube-example
spec:
  selector:
    app: kube-example
  ports:
    - port: 80
      targetPort: 3000
  type: LoadBalancer

and set all urls in the env list with image policy
